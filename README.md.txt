# 🎓 Risk2Rise
AI-Powered Student Risk Prediction and Early Intervention System

## 📌 Overview

Risk2Rise is an intelligent student monitoring system that predicts academic risk levels using Machine Learning and provides early intervention support.

It helps teachers identify at-risk students early and take preventive action.

---

## 🚀 Features

• Student Login System  
• AI Risk Prediction (Low, Medium, High)  
• Interactive Dashboard  
• Attendance Visualization  
• Marks Analysis  
• Risk Meter  
• Email Alert System  
• Teacher Dashboard  

---

## 🧠 Machine Learning

Algorithm Used:

Random Forest Classifier

Accuracy:

83.5%
